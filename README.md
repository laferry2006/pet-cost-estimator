# 🐾 宠物养宠开销估算器

基于真实宠物店销售数据的交互式开销估算工具，帮助想要养宠物的人了解养宠的各项开销。

## 📊 数据来源
- **数据集**: Pet Store Records 2020 (Kaggle)
- **数据链接**: https://www.kaggle.com/datasets/ippudkiippude/pet-store-records-2020
- **数据时间**: 2020年

## 🚀 快速开始

### 本地运行

1. 安装依赖
```bash
pip install -r requirements.txt
```

2. 运行应用
```bash
streamlit run app.py
```

3. 在浏览器中打开 http://localhost:8501

## ✨ 功能特点

### 1. 宠物类型选择
支持6种宠物类型：
- 🐱 猫咪 - 独立优雅，适合上班族
- 🐶 狗狗 - 忠诚活泼，需要陪伴
- 🐠 鱼类 - 安静省心，适合新手
- 🦜 鸟类 - 聪明可爱，能学说话
- 🐹 仓鼠 - 小巧可爱，成本低
- 🐰 兔子 - 温顺乖巧，需要空间

### 2. 地区选择
支持9个国家/地区：
- 🇮🇳 印度
- 🇺🇸 美国
- 🇸🇬 新加坡
- 🇧🇪 比利时
- 🇩🇪 德国
- 🇹🇷 土耳其
- 🇻🇳 越南
- 🇱🇰 斯里兰卡
- 🇯🇵 日本

### 3. 开销类别
从四个方面估算开销：
- 🛁 **洗护** - 洗澡、美容、清洁用品
- 🍖 **食物** - 主粮、零食、营养品
- 🎾 **玩具及工具** - 玩具、笼子、食盆等
- 💊 **医疗** - 疫苗、体检、药品

每个类别可选择三个需求等级：
- **基础** - 基本需求，价格 × 0.7
- **标准** - 正常需求，价格 × 1.0
- **高级** - 高端需求，价格 × 1.5

### 4. 可视化展示
- 📊 开销明细表格
- 🥧 开销分布饼图
- 📊 各类别对比条形图
- 💡 养宠建议

## 📈 分析结果亮点

### 关键发现
- **总销售量**: 102,047单位
- **总收入**: ₹10.14亿
- **平均评分**: 6.5/10
- **复购率**: 46.6%

### 各类别表现
| 类别 | 最高收入 | 最高复购率 | 最高评分 |
|------|---------|-----------|---------|
| 食物 | Snack | - | Food |
| 洗护 | - | - | - |
| 玩具及工具 | Equipment | - | - |
| 医疗 | Medicine | - | - |

### 地区分布
- **印度**占65%市场份额
- 其他主要市场：美国、新加坡、德国等

## 🛠️ 技术栈

## 📝 使用说明

1. 在左侧边栏选择宠物类型
2. 选择所在地区
3. 调整四个类别的需求等级
4. 点击"计算开销"按钮
5. 查看详细的开销估算和可视化图表
